# Al-Quds Lana Mobile Store Website
An e-commerce and service platform for Al-Quds Lana Mobile Shop, Zarqa, Jordan.
---
## Overview
Al-Quds Lana's website is a dual-purpose platform built to modernize the in-store experience and extend it online. Whether a customer is looking to browse the latest phones or book a repair, this site handles both.
Two core functions:
- **Online Product Showcase**: Browse and explore available phones and accessories
- **Maintenance Ticket Booking**: Submit and track repair service requests
---
## About Al-Quds Lana
Al-Quds Lana is a mobile phone shop based in Zarqa, Jordan, offering device sales and repair services. This website is built to improve the shopping and service experience for its customers.
